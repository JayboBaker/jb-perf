# Movie search facility

## Overview
A single-page app built using [react](https://github.com/facebook/react)

## Getting Started
* Install [Node.js](https://nodejs.org/en/) (LTS version 6 is recommended), this should include NPM

* Run `npm i`

* Run `npm start`

## Running tests
Run `npm test`

## Watching tests
Run `npm run test:watch`

## Production builds
Run `npm run build`
